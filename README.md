# Kalimati Analysis and Prediction

This project provides tools and workflows for analyzing and predicting vegetable prices using data from the Kalimati market. It includes data cleaning, exploratory data analysis (EDA), and machine learning models for forecasting.

## Project Structure

```
kalimati-analysis-and-prediction/
├── main.py                  # Entry point script
├── pyproject.toml           # Project metadata and configuration
├── requirements.txt         # Python dependencies
├── data/
│   ├── raw/                 # Raw data files
│   └── processed/           # Cleaned and processed data
├── docs/                    # Documentation
├── models/                  # Saved models and artifacts
├── notebooks/               # Jupyter notebooks for EDA, modeling, and analysis
├── references/              # Reference materials
├── reports/
│   └── figures/             # Generated figures and plots
├── src/                     # Project Pipeline & Inference 
│   
│   
├── tests/                   # Test scripts
├── utils/                   # Utility modules
```

## Data
Place raw data in the `data/raw/` directory. Processed data will be stored in `data/processed/`.
